Enchantify Inside Cover iOS Widget

This is a small SwiftUI + WidgetKit companion app. The widget reads a compact
state file exported by:

  python3 scripts/widget-state.py bj

The exporter writes:

  hooks/widget-state.json
  hooks/widget-image.png

The JSON embeds the image data, so the iOS app only needs to import the JSON.

Install path:

1. Open ios/InsideCover/EnchantifyInsideCover.xcodeproj in Xcode.
2. Select the InsideCoverApp target, set your Team, and change the bundle id if
   Xcode asks.
3. Select the InsideCoverWidgetExtension target and use the same Team.
4. In Signing & Capabilities, make sure both targets share the same App Group:
   group.com.openclaw.enchantify.insidecover
5. Run InsideCoverApp on the iPhone or iPad.
6. On the device, open the app and import hooks/widget-state.json by AirDrop,
   iCloud Drive, or Files.
7. Add the "Enchantify Inside Cover" widget from the Home Screen widget picker.

Refresh path:

1. Run python3 scripts/widget-state.py bj on the Mac.
2. Send hooks/widget-state.json to the device.
3. Open it in InsideCoverApp. The widget reloads.

Prototype note:

This is intentionally local and private. Later, the app can fetch the widget
state from iCloud Drive or a private endpoint so manual import is no longer
needed.
