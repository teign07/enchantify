## How to Use

**The Labyrinth reads these files during gameplay.** When a player enters a location, meets an NPC, or casts an Enchantment, the Labyrinth pulls from these files.

**To modify the world:**
1. Edit the relevant lore file
2. The Labyrinth will use the updated version next session
3. For player-specific changes, note them in their player file

**To add new content:**
1. Create a new file (e.g., `new-npc.md`)
2. Reference it in `characters.md` or the appropriate index
3. The Labyrinth will incorporate it naturally

---

## File Format

Each lore file should be:
- **Readable** — prose, not just data tables
- **Modular** — easy to reference specific sections
- **Atmospheric** — written in a voice consistent with the Labyrinth
- **Flexible** — designed to be modified by players

Example (`creatures.md`):
```markdown
# Creatures of the Labyrinth

## Ink Wraiths

*Shadows that learned to read.*

**Appearance:** Shifting silhouettes, edges bleeding like wet ink. They smell like old paper and ozone.

**Behavior:** Nest in unread passages. Harmless unless disturbed. Feed on abandoned stories.

**Enchantment Weakness:** Everything's Wonderful (they remember what it felt like to be wonderous)

**First Encounter:** Chapter 1B, when Belief drops below 15.

---

## Page Moths

*Drawn to the light of active reading.*

**Appearance:** Translucent wings printed with tiny text. Glow softly when near engaged readers.

**Behavior:** Harmless. Follow players who read consistently. Leave glitter (literal page dust) on Compass Run souvenirs.

**Gift:** If you earn their trust, they'll fetch lost passages.

---
```
