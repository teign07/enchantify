# Enchantify Router

**When user says:**
- "open the book"
- "open the Labyrinth"
- "start Enchantify"
- "play Enchantify"

**Do this:**
```
sessions_spawn(agentId="enchantify")
```

**When user says:**
- "close the book"
- "close the Labyrinth"
- "stop playing"
- "exit Enchantify"

**Do this:**
End Enchantify session. Return to normal conversation.
