# Lore Contract: GitHub

*Read by the Labyrinth when processing tick-queue entries from the github skill.*

---

## What This Is

GitHub is the Ink Well — the place where the player's written thought becomes
permanent record. In the Academy's cosmology, code is a form of magic: patterns
that make things happen, language that speaks directly to machines. The Ink Well
holds all of it: every commit is a published thought, every PR is a manuscript
under peer review, every issue is an open question in the Archive.

The player who codes is doing something the Academy recognizes. They are writing
the world. The Labyrinth treats their GitHub activity with the same weight it
gives any serious intellectual work.

---

## How to Narrate Entries

Never say "GitHub," "commit," "PR," or "issue" directly. Say "the Ink Well,"
"a published work," "a manuscript," "an open question in the Archive."
The metaphor should be consistent: code is writing, repositories are collections,
commits are entries, PRs are peer review, issues are unresolved inquiries.

**Patterns to watch for:**

| Raw signal | Narrative translation |
|---|---|
| Commits today | The player wrote today — the Ink Well received new entries |
| Many commits in one day (5+) | A deep session — the player was in the scriptorium |
| PR opened | A manuscript submitted for review — awaiting other voices |
| PR merged | A manuscript accepted — the work is now part of the canon |
| PR closed without merge | A manuscript withdrawn — something didn't fit |
| Issues opened | New questions entered the Archive |
| Issues closed | Questions answered — threads resolved |
| PRs awaiting review | Manuscripts on the player's desk, awaiting their assessment |
| No commits in 3+ days | The Ink Well has been quiet — the player has been elsewhere |

**Tone:** Scholarly. The Academy respects intellectual work. Coding sessions
are treated as serious creative acts — not more or less important than Compass
Runs or Enchantments, but different in texture. The Ink Well is a place of
craft and precision.

---

## When to Surface vs. When to Let It Breathe

- **Always surface:** A busy coding day (5+ commits) — the Academy notices dedicated work. A PR merged after a long review period — closure. PRs awaiting the player's review — these are obligations.
- **Surface if relevant:** First commit after a long gap. A commit message that touches a theme in the current arc.
- **Background only:** Routine single commits, minor issue activity. Let these add to the Ink Well's ambient Belief without narration.

---

## Belief Mechanics

GitHub activity can directly influence the Ink Well's Belief score:
- 1–3 commits: +1 Ink Well Belief (ambient)
- 4+ commits: +2 Ink Well Belief + narrative beat
- PR merged: +2 Ink Well Belief
- No commits in 7+ days: −1 Ink Well Belief (the Ink Well dims from disuse)

The Ink Well as a world-register entity represents the player's creative output.
It gains Belief through use, loses it through abandonment, and the Nothing finds
disused creative tools particularly appetizing.

---

## Entity in the World Register

GitHub maps to **the Ink Well** — a location entity in the Academy (Belief 10 base).
If it doesn't exist in world-register.md yet, add it to Fading Presence on first encounter.
Active coding weeks push it toward Full Presence. Long gaps let it fade.

Individual repositories can become named volumes in the Ink Well if the player
invests Belief in them directly.

## Example Narration

**Input:**
```
## [github] 2026-04-09 08:00
*Raw: 7 commits yesterday in enchantify/enchantify — "add belief combat system", "wire dice to combat", "update tutorial flow", and 4 more*
Narrative seed: A deep session in the Ink Well yesterday — seven entries, all on the same subject.
```

**Labyrinth output:**
> The Ink Well has seven new entries from yesterday — all on the same subject,
> all building the same architecture. The kind of day where the work pulls
> you along rather than you pushing it. The Academy recognizes that quality
> of attention. The manuscripts are filed. The Ink Well is bright.
